# Web Testing Project

Практический проект по ручному тестированию веб-приложения (интернет-магазин).

## Цель проекта
Получение практического опыта:
- написание тест-кейсов
- поиск багов
- проверка пользовательских сценариев

## Что было сделано
- Проверка форм (логин, регистрация)
- Проверка корзины
- Написание тест-кейсов
- Оформление баг-репортов

## Инструменты
- Git
- GitHub
- DevTools

## Структура
- test-cases
- bug-reports
